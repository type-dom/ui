## 项目说明

> 这是一个基于自建前端框架TypeDom创建的ui组件库。

## 项目环境

## 项目运行

> 1、安装环境：yarn install

> 2、运行项目： yarn serve

> 3、生成js版本项目代码: yarn js

> 4、打包项目： yarn build

> 5、生成库： yarn lib

> 6、生成类图： yarn uml

> 7、生成文档：yarn docs

> 8、代码检查：yarn ts-coverage
